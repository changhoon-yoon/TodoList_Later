🧭 참고 - PCANtool 프로젝트 맥락

이 프로젝트는 소스 저장소가 아니라 키패드 CAN 통신 도구의 배포판 + 필드 테스트 환경. 실행 파일은 pcan_125k/250k/500k.exe로 baud rate별 분리 배포. Windows 전용.
핵심 설정은 config.ini. J1939 프로토콜 기반, CAN ID는 HEX 29-bit extended 형식. 키패드 주소는 거래처별로 다름 (예: 우진산전 = 0x21).
